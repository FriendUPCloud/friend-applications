Google Docs

A web application to access the external Google Doc word processor in a Friend Window

  • A valid Google account is required to access the Google word processor. Click here to go to Google and create a new account.
  • Files are automatically stored to your Google Drive, not your Friend drive.
  • To enable Google Drive support in Friend 1.2, Navigate to the System menu and select My account. On the Storage tab, add a Google drive and authorize it in Friend

Limitations

  • This app will only create NEW documents. It will not open existing documents via the File->Open method. To open an existing document, double click the file directly from your Drive folder in the file manager.
  • This app will open new documents or template (both from the File menu) in a new tab.
  • This app will not access any Google service such as Photos, Drive files, etc. To insert any images or other items into a document, the file must reside on the local machine.
